# 🚀 Alpha-Soft Task Tracker API

REST API для управления проектами и задачами, разработанный на **Django** и **Django REST Framework**.
Проект реализует ролевую модель доступа, JWT-аутентификацию, фильтрацию данных и автоматическую документацию.

## ⚡ Функционал
- **Управление проектами:** Создание, редактирование, удаление проектов. Автоматическое назначение создателя.
- **Управление задачами:** CRUD операции. Автоматическое назначение автора.
- **Ролевая модель (Permissions):**
  - **Владелец проекта:** Полный доступ.
  - **Участник:** Просмотр своих проектов и задач.
  - **Исполнитель (Assignee):** Может менять только статус и приоритет задачи.
  - **Автор:** Может редактировать описание.
- **Валидация:** Проверка, что исполнитель является участником проекта.
- **Фильтрация и поиск:** По статусу, приоритету, дедлайну, исполнителю.
- **Аутентификация:** JWT Token (Access + Refresh).
